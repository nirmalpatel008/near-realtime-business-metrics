# Sample Natural Language Questions

Use these for Amazon Q in QuickSight or the local query catalog runner.

## Executive

```text
What was DAU yesterday?
Show DAU for the last 7 days.
What was the conversion rate this week?
How fresh is the dashboard?
```

## Funnel

```text
Show the funnel for yesterday.
Where are users dropping off?
How many sessions reached payment initiated?
How many sessions completed a purchase?
```

## Payments

```text
What was the payment success rate yesterday?
Show failed payments by reason.
Which payment method has the lowest success rate?
Are payment failures increasing?
```

## Geography

```text
Which city had the most active users?
Which city had the lowest payment success rate?
Show payment success rate by city.
Which region should we investigate?
```

## Devices

```text
Compare conversion rate by device type.
Is Android conversion lower than iOS?
Show payment success rate by operating system.
Which device type has the most failed payments?
```

## Portfolio Demo Sequence

Use this sequence in a 3-5 minute demo:

```text
What was DAU yesterday?
Show conversion rate by device type for the last 7 days.
Which city had the lowest payment success rate?
Show failed payments by reason.
How fresh is the dashboard?
```
